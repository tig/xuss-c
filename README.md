# Xuss-C

Xuss-C is a pocket companion for the M5Stack M5GO: a living face, a short boot riff, full-song playback of *First*, and a built-in sensor Details screen.

Same product as [tig/xuss](https://github.com/tig/xuss), rebuilt in **native C on ESP-IDF** so audio stays clean and the UI stays lively while music plays — overcoming the MicroPython twin’s weak multitasking and poor audio path.

The name is the short half of Turminder Xuss, the drone in Iain M. Banks' *Matter*: precise dirty work while the human keeps judgment.

## Status

**Spec only.** [spec.md](spec.md) is the contract. The firmware gets built from it by an agent, on a branch, with an ambiguity log, using ESP-IDF.

Hardware is an M5Stack M5GO IoT Starter Kit v2.7.

See also: [tig/xuss](https://github.com/tig/xuss) (MicroPython twin), [silico](https://github.com/tig/silico).
