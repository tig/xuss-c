# Xuss-C

Xuss-C is a test instrument (engine-speed edge trains, input forcing, current measurement) for **C / ESP-IDF** embedded device development.

Same product as [tig/xuss](https://github.com/tig/xuss); the only intentional difference is native C on ESP-IDF instead of MicroPython on [silico](https://github.com/tig/silico).

The name is the short half of Turminder Xuss, the drone in Iain M. Banks' *Matter*: precise dirty work while the human keeps judgment.

## Status

**Spec only.** [spec.md](spec.md) is the contract. The firmware gets built from it by an agent, on a branch, with an ambiguity log, using ESP-IDF.

Hardware is an M5Stack M5GO IoT Starter Kit v2.7.

See also: [tig/xuss](https://github.com/tig/xuss) (Python twin), [silico#53](https://github.com/tig/silico/issues/53).
